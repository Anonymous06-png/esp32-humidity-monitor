# ESP32 Humidity Monitor (HTTP)

## Description
This project uses ESP32 to monitor temperature and humidity,
display data on OLED, and send sensor data to a server using HTTP.

## Features
- Read temperature & humidity (AHT30)
- OLED display
- Alert using LED, relay, buzzer
- Send data via HTTP (JSON)

- HTTP
